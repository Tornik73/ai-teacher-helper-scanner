## Getting Started

### Step 1: Read Documentation

- Start with [QUICKSTART.md](QUICKSTART.md) for setup
- Skim [README.md](README.md) for overview

### Step 2: Install & Build

```bash
npm install
npm run build
```

### Step 3: Load Extension

1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select `dist/` folder
5. ‚úÖ Done!

### Step 4: Test

1. Visit any Quizlet set page
2. Click extension icon
3. Should show detected cards
4. Try editing and exporting

### Step 5: Explore Code

- Read comments in [src/services/quizletService.ts](src/services/quizletService.ts) to understand extraction
- Check [src/popup/popup.tsx](src/popup/popup.tsx) for UI implementation

## Extension Loading in Chrome

### What Chrome Reads

1. `dist/manifest.json` - Extension config & permissions
2. `dist/background.js` - Service worker
3. `dist/content.js` - Content script (injected on Quizlet pages)
4. `dist/popup.html` - Popup UI
5. `dist/popup.js` - React app (loaded by popup.html)
6. `dist/images/` - Icons

### What You Modify

- Source files in `src/` folder
- Run build process
- Reload extension
- **Don't** modify `dist/` directly
